# Simple Calculator

A lightweight calculator built with **HTML, CSS, and JavaScript**. Supports basic arithmetic operations (+, -, *, /, %) and decimals with a clean digital-style interface.  

## Features
- Addition, subtraction, multiplication, division, percentage  
- Decimal input and clear (`AC`)  
- Responsive button layout  

## Usage
Open `index.html` in a browser, click numbers/operators, press `=` to calculate, `AC` to clear.
